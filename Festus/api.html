API Overview

Access open government data through RESTful endpoints designed for citizen engagement, analytics and reporting.

Authentication

Register for an API key and use Bearer tokens with each request. Rate limits apply for public access tiers.

Code Examples

JavaScript

fetch('/api/datasets', { headers: { Authorization: 'Bearer YOUR_API_KEY' } })
  .then(res => res.json())
  .then(data => console.log(data));
API Keys

Manage your API keys in the portal and track usage limits per key.

Rate Limits
  • 1000 requests/day
  • 60 requests/minute
  • Public access tier only
