About the Right to Access Information Commission

HomeAbout RAIC

Mandate and mission

The Right to Access Information Commission (RAIC) is responsible for implementing the Access to Information Act and building an open data ecosystem that empowers citizens through transparency, accountability and participation.

Core objectives

  • • Promote public access to government information.
  • • Enable data-driven decision making.
  • • Monitor government performance through disclosure.
  • • Support citizen engagement and feedback.
Key statistics

Institutional metrics for the RAIC open data program.

20+ Active ministries publishing datasets
150+ Published transparency reports
95% Response rate to information requests
Governance Principles

Open data is guided by these principles.

  1. Open by default
  2. Machine-readable
  3. Non-discriminatory access
  4. Timely updates
RAIC Strategy

Expand open data coverage across government functions with a user-centered portal, API access, and clear information request workflows.

Public Participation

Citizens can request information, review published budgets, and submit feedback through a dedicated information request portal.

About RAIC

RAIC Sierra Leone ensures public access rights and supports a strong open data environment for accountable governance.
